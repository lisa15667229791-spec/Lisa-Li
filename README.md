# Lisa Academic Website Starter

This is a simple one-page academic website.

## Files

- `index.html`: webpage content
- `style.css`: webpage design

## How to view it

1. Download and unzip the folder.
2. Open `index.html` in your browser.

## How to edit it

Open `index.html` with any text editor, such as VS Code, TextEdit, or Notepad.

Things to change first:

1. Replace `Lisa` with your full professional name if you want.
2. Replace `your-email@example.com` with your email.
3. Replace the photo placeholder with your image.
4. Add your CV PDF link.
5. Add your real publications, presentations, and projects.

## How to publish with GitHub Pages

1. Create a GitHub account.
2. Create a new repository named `yourusername.github.io`.
3. Upload `index.html` and `style.css`.
4. Go to Settings > Pages.
5. Choose the main branch.
6. Your website will be available at `https://yourusername.github.io`.
